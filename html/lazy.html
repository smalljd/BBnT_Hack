Fuel Hackathon API

User Count:100 Accounts Count: 342 Transactions Count: 545084

/users
Get all users.
curl -v http://fuelhackathon.herokuapp.com/users -H 'accept: application/json'

/user/{id}
Get one user for a given userId.
curl -v http://fuelhackathon.herokuapp.com/user/54dc263b1a0bb10c0025e031 -H 'accept: application/json'

/accounts/{id}
Get all accounts for a given userId.
curl -v http://fuelhackathon.herokuapp.com/accounts/54dc263b1a0bb10c0025e031 -H 'accept: application/json'

/transactions/{id}
Get all transactions for a given accountId.
curl -v http://fuelhackathon.herokuapp.com/transactions/54dc263b1a0bb10c0025e032 -H 'accept: application/json'

/transactions/{id}/{year}/{month}
Get a set of transactions for a given account for a specific month (note if month doesn't have transactions this returns empty array
curl -v http://fuelhackathon.herokuapp.com/transactions/54dc263b1a0bb10c0025e032/2012/08 -H 'accept: application/json'

/transaction/{id}
Get a single transaction for a given transactionId.
curl -v http://fuelhackathon.herokuapp.com/transaction/54dc263c1a0bb10c0025e039 -H 'accept: application/json'

/counts
Get the counts and the number of transactions in the apis are updated along with the data every 3600 seconds
curl -v http://fuelhackathon.herokuapp.com/counts -H 'accept: application/json'

/hits
Get the number of hits to the apis from the last server restart
curl -v http://fuelhackathon.herokuapp.com/hits -H 'accept: application/json'
